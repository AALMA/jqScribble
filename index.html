jqScribble by jimdoescode

A touch enabled jquery plugin for drawing on a canvas

A jquery plugin that will create a canvas element that you can then scribble on by dragging your mouse, or your finger, across. It also supports displaying an image that can be drawn on and provides a way to hook into saving your image so that it might be passed to a server.You can also easily create new brush types for it that can be passed to it.

Dependencies

jQuery javascript library

Execute

$("#test").jqScribble({
	width: 			350px,	//If not defined is taken from the attached element
	height: 		250px,	//If not defined is taken from the attached element
	backgroundImage: 	false, 	//Define a path to an image to draw on.
	backgroundImageX: 	0, 		//Offsets background image in x direction
	backgroundImageY: 	0,		//Offsets background image in y direction
	saveMimeType: 		"image/png", 		//Save image as mimeType
	saveFunction: 		BasicCanvasSave,	//Lets you define your own save function for saving to a server or something
	brush:			BasicBrush,		//Lets you define a custom brush
	brushSize:		2,			//Increase or decrease the size of the brush
	brushColor:		"rgb(0,0,0)"		//Change the color of the brush.
});
